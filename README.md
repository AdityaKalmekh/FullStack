# Full-Stack Assessment

## Setup Instructions

### 1. Install Dependencies

**Backend:**
```bash
cd server
npm install
```

**Frontend:**
```bash
cd client
npm install
```

### 2. Run the Application

**Terminal 1 - Start Backend (Port 5000):**
```bash
cd server
npm start
```

**Terminal 2 - Start Frontend (Port 5173):**
```bash
cd client
npm run dev
```
